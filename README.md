### 🚀 Projetos em destaque

**[TUPÃ — Base de Referência Ambiental](https://github.com/Cyborgdsi47)** *(haCARthon ENAP / Dataprev / SFB)*
* Plataforma web full-stack que automatiza a geração de bases de referência ambiental para imóveis rurais (CAR).
* Cruza dados de satélite e relevo com o Código Florestal via geoprocessamento espacial em PostGIS, entregando diagnósticos automáticos.
* **Stack:** FastAPI, Python, PostgreSQL 17, PostGIS, Supabase, React 18, TypeScript, Vite.

**[SafraSense — Monitoramento Agrícola por Satélite](https://github.com/Cyborgdsi47)** *(5º Lugar - Hackathon CopernicusLAC 2026)*
* Plataforma web que democratiza o acesso a seguros agrícolas para produtores rurais, calculando o índice NDVI a partir de imagens reais do satélite Sentinel-2 da ESA.
* Gera um laudo satelital em PDF com histórico para validação de seguradoras.
* **Stack:** FastAPI, Python, GeoPandas, NumPy, React 19, TypeScript, Firebase, Leaflet, jsPDF.

**[RemedioSP](https://github.com/Cyborgdsi47/Rem-diosp)**
* App web (Next.js + Supabase) para buscar remédios com preço real e encontrar a farmácia mais próxima.
* Catálogo baseado na tabela CMED com busca full-text, comparação de preços por rede e lista de favoritos.
* **Stack:** Next.js, Supabase, APIs públicas.
